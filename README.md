# TP AlgoBox – Travail collaboratif avec GitHub

## Description
Ce projet a été réalisé dans le cadre d’un TP visant à découvrir l’utilisation de GitHub
comme outil collaboratif pour un projet AlgoBox.
Il contient un exemple de fichier AlgoBox (.algo) et illustre l’utilisation des commandes Git.

## Organisation du projet
- src/ : dossier contenant les fichiers sources AlgoBox (.algo)
  - exemple.algo : fichier AlgoBox du projet

## Prérequis
- Logiciel AlgoBox installé sur l’ordinateur
- Git installé (optionnel pour l’exécution)

## Ouverture et exécution
1. Ouvrir le logiciel AlgoBox
2. Cliquer sur **Fichier → Ouvrir**
3. Aller dans le dossier `src/`
4. Sélectionner le fichier `exemple.algo`
5. Cliquer sur **Exécuter**
